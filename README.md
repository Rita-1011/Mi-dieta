# Mi Diet Nutrition Planner

Aplicación web para importar, organizar y gestionar planes nutricionales semanales.

## Funcionalidades

- Inicio de sesión con Supabase Authentication.
- Importación de dietas en formato texto.
- Parser automático de planes nutricionales.
- Organización por días de la semana.
- Edición y eliminación de comidas.
- Lista de la compra generada automáticamente.
- Asistente IA.
- Diseño responsive.

## Tecnologías

- HTML
- CSS
- JavaScript
- Supabase
- Bolt.new
