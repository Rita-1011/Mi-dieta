# Mi Dieta

Aplicación web para la gestión inteligente de planes nutricionales mediante Inteligencia Artificial.

## Descripción

Mi Dieta permite importar planes nutricionales, analizarlos mediante Inteligencia Artificial y organizarlos automáticamente por días y comidas. Además, incorpora la generación automática de la lista de la compra y un asistente nutricional para resolver dudas y proponer sustituciones de alimentos.

## Funcionalidades principales

- Autenticación de usuarios mediante Supabase Authentication.
- Importación y análisis de planes nutricionales mediante Inteligencia Artificial.
- Organización automática de comidas por días de la semana.
- Gestión de comidas (consulta, edición y eliminación).
- Generación automática de la lista de la compra.
- Asistente nutricional basado en Google Gemini.
- Diseño responsive para dispositivos móviles y escritorio.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Supabase (Authentication, PostgreSQL y Edge Functions)
- Google Gemini 2.5 Flash
- Netlify
